
# Lumir Bot Installer
Installer Bat file for lumir bot

## MetaTrader 4

**What will install**
| File name | Modifile | Used for |
|--|--|--|
| setup/experts.ini | Instance/config/experts.ini |Modify the list of domain addresses that the terminal allows to send data to|
| LumirMT4_v2.ex4 | Instance/MQL4/Experts | Copy the bot file to the terminal |
| -- history -- | Instance/config/terminal.ini | Modification allows to retrieve data of entire transaction history |

**How to use**

 1. Run install.bat file and chose broker terminal you want to install
 2. Chose broker terminal to install
